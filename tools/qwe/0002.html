2.HTML教程-各种样式的文字

简介说完了，那么来看看怎么制作网页吧（这期只说说html文字的样式）

你需要：
  • 一个html编辑器
  • 一个浏览器（如IE 8.0，chrome等）

先来看看各种的文字效果：

This is a text
This is a text
This is a text
This is a text
This is a text
This is a text
这些文字效果的代码分别是：<a>就是普通文字，<em>代表斜体文字，<strong>粗体文字，<s>则是文字带上删除线，<u>是文字带下划线。此外还有各种html的各种标题代码，分别是<h1>（大标题）<h2>（较大标题）<h3>（中等标题），一直到h6就没了。还有<br>和<hr>代表换行。接着你可以新建一个文件，保存为"1.html",然后试试以下代码：
<meta charset='UTF-8'>
<h1>这是文字</h1>
<h2>这是文字</h2>
<h3>这是文字</h3>
<p>这是文字</p>
<em>text</em>
<s>这是文字</s>
<u>这是文字</u>
<strong>文字</strong>
<br>
<a>----OK----</a>
